# ClearNight — Marketing Site

The marketing and support site for **ClearNight**, an iOS app that forecasts stargazing conditions based on cloud cover, moon phase, and dark-sky quality.

Live at [clearnightapp.com](https://clearnightapp.com).

## About

A lightweight, static site built as the public front door for the app — landing page, privacy policy, terms, and support. Designed to match the app’s dark, night-sky aesthetic.

## Pages

- **index.html** — Landing page with app overview and screenshots
- **privacy.html** — Privacy policy
- **terms.html** — Terms of use
- **support.html** — Support and FAQ

## Stack

Hand-written HTML and CSS, no framework. Hosted on GitHub Pages via a custom domain (`CNAME`). Typeface: Space Grotesk.
